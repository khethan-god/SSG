# Static Site Generator

## Description
This project, created during the Boot.dev curriculum, converts markdown files into HTML web pages. Users can write their webpage content in markdown format, which is more intuitive than writing HTML directly. The `main()` function processes files from a source directory containing markdown files and generates corresponding HTML files in a destination directory.

## Installation
1. Ensure Python is installed on your system
2. Clone this repository:
```bash
git clone https://github.com/khethan-god/SSG.git
```

3. Give execute permission to `main.sh` if not given already:
```bash
chmod +x main.sh
```

## Usage
1. Place your markdown files in the `static` directory
2. Run the conversion script:
```bash
./main.sh
```
